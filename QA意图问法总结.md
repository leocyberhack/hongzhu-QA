# QA意图问法总结

## 景区QA

- 意图数：10
- 具体问法数：53

### 1. 门票与优惠政策

1. 1.1 当前门票/套餐的水牌/市面价格是多少，包含哪些类型的产品/票种？
2. 1.2 学生/老人/儿童/特殊人群有哪些优惠或免票条件？
3. 1.3 线上购票和现场购票有什么差异，套票包含哪些项目？
4. 1.4 优惠资格现场核验失败时，去哪里复核、需要哪些凭证？

### 2. 开放时间与入园时段

1. 2.1 景区开闭园时间和最晚入园时间分别是几点？
2. 2.2 节假日或夜场的入园时段是否有变化？
3. 2.3 园内项目通常会提前多久停止排队或检票？
4. 2.4 遇到恶劣天气或临时闭园时，已购门票和项目如何退改？

### 3. 交通到达与停车

1. 3.1 乘坐公交/地铁/高铁到景区的推荐路线是什么？
2. 3.2 停车场在哪里、收费多少、是否需要预约车位？
3. 3.3 打车到景区大约费用多少，哪个入口下车更方便？
4. 3.4 到场后走错门或停车场满位时，改走哪个入口、是否有接驳？
5. 3.5 市区/县城到景区大概有多远，通常车程多久？
6. 3.6 从火车站/机场到景区大概有多远，常见到达方式有哪些？

### 4. 取票核销与实名入园

1. 4.1 线上购票后入园是刷身份证、刷码还是需要换票？
2. 4.2 电子身份证、港澳台或其他证件能否直接核验入园？
3. 4.3 一个订单多张票能否分批入园，同行人必须同时进吗？
4. 4.4 闸机提示“已使用/无效/不匹配”时，如何走人工通道处理？

### 5. 游玩路线与项目排队

1. 5.1 半天或一天游玩分别推荐什么路线？
2. 5.2 必看景点和演出的时间、位置怎么安排最省时？
3. 5.3 是否有观光车/导游服务，能否查看实时排队信息？
4. 5.5 景区的核心亮点/必打卡吸引点有哪些？
5. 5.6 第一次去这个景区，最推荐的玩法组合是什么？
6. 5.7 想拍照+轻松逛，有没有体力消耗较小的路线？
7. 5.8 景区大概需要玩多久，半天/一天怎么安排？
8. 5.9 周末和小长假人流大概什么程度，哪些时段最拥挤？

### 6. 费用包含与二次消费

1. 6.1 门票是否包含索道、游船、观光车等项目？
2. 6.2 套票具体包含哪些内容，哪些项目需要另付费？
3. 6.3 保险、讲解器、二次消费项目的收费标准在哪里看？
4. 6.4 现场出现额外收费争议时，去哪里核验价格并申请处理？
5. 6.5 景区有没有乱收费或强制消费，遇到了怎么维权？

### 7. 退改签与订单变更

1. 7.1 临时去不了时，门票是否可退、最晚何时申请？
2. 7.2 退票手续费怎么收，退款大概多久到账？
3. 7.3 买错日期或行程变化时，如何改期改签？
4. 7.4 退改被拒或平台找不到入口时，如何提交人工复核？

### 8. 基础设施与便民服务

1. 8.1 园内餐饮、饮水补给点分布和价格大致如何？
2. 8.2 行李寄存、厕所、医务室和游客中心分别在哪里？
3. 8.3 是否提供轮椅/婴儿车、充电宝、WiFi，信号是否稳定？
4. 8.4 充电宝借还失败或寄存柜打不开时，现场找谁处理？
5. 8.5 景区路况难不难走？推婴儿车方便吗？老人能不能走下来？
6. 8.6 园内/周边有什么必吃小吃和特色菜，哪些店容易踩雷要避开？

### 9. 携带物品与安全限制

1. 9.1 可以携带哪些物品入园（食物、宠物等）？
2. 9.2 无人机、自拍杆等拍摄设备使用规则是什么？
3. 9.3 孕妇、心脏病、恐高等风险人群参与项目有哪些限制？
4. 9.4 现场被劝退或限制参与项目时，能否改项目或退差价？

### 10. 周边配套与异常协同

1. 10.1 景区周边住宿和餐饮如何选，玩两天住哪里更方便？
2. 10.2 晚间出园后打车和返程交通是否方便？
3. 10.3 临时停运、限流、分时放行等公告在哪里看最准确？
4. 10.4 出园后发现物品遗失或行程中断时，如何联系景区协助处理？
5. 10.5 这个景点真实体验怎么样，会不会过度商业化或踩坑？
6. 10.6 实景和小红书照片差别大吗，会不会“照骗”？
7. 10.7 景区官方联系方式是什么（客服电话/公众号/官网）？
8. 10.8 想拍照的话，景区哪些机位最出片？

## 酒店QA

- 意图数：10
- 具体问法数：50

### 1. 房型细节与入住体验

1. 1.1 这个房型的大床/双床尺寸分别是多少，能否拼床？
2. 1.2 房间有窗吗，面积多大，通常安排在几楼？
3. 1.3 房间是否配浴缸、阳台、冰箱和保险柜？
4. 1.4 到店发现房型与预订描述不符（无窗/面积偏差/景观不符）怎么办，能否换房或补差？
5. 1.5 这家酒店的核心亮点/吸引点是什么，适合哪类人群？
6. 1.6 卫生和宣传照片一致吗，有没有霉味、污渍等常见问题？

### 2. 入住与退房流程

1. 2.1 最早几点可以办理入住，最晚几点退房？
2. 2.2 需要提前入住或延迟退房时怎么收费？
3. 2.3 凌晨到店、提前到店寄存行李、入住证件要求分别是什么？
4. 2.4 前台提示证件不合规或人证不一致无法入住时，去哪里处理、需要什么材料？
5. 2.5 行李寄存是否方便（到店前/退房后），流程和时效如何？

### 3. 早餐餐饮与外卖

1. 3.1 房费包含几份早餐，早餐是自助还是定食？
2. 3.2 儿童早餐怎么收费，身高/年龄免费标准是什么？
3. 3.3 酒店支持外卖送上楼或送餐服务吗？
4. 3.4 出现漏餐/错餐/食品安全问题时，如何补送或退款，处理时效多久？

### 4. 儿童与加床政策

1. 4.1 房间可以加床吗，费用多少，是否有限量？
2. 4.2 儿童不占床怎么收费，会影响早餐和用品吗？
3. 4.3 酒店是否提供婴儿床、亲子设施、儿童餐椅等服务？
4. 4.4 现场对儿童年龄/身高判定有争议或未成年人入住受限时，如何复核？

### 5. 交通接送与停车

1. 5.1 酒店是否提供接送机服务，最晚需提前多久预约？
2. 5.2 自驾停车是否免费，车位在地上还是地下，是否需要预约？
3. 5.3 停车场有充电桩/代客泊车/景区班车吗？
4. 5.4 接送机预约失败、车位已满或充电桩故障时，替代方案和补偿规则是什么？
5. 5.5 酒店到市区/县城大概有多远，常见通勤方式和车程多久？
6. 5.6 酒店到附近热门景点大概多远，打车/公交方便吗？
7. 5.7 从火车站/机场到酒店大概多远，推荐怎么过去？

### 6. 退改与订单变更

1. 6.1 最晚什么时候可以免费取消订单？
2. 6.2 不可取消、不可抗力、买错日期分别如何退改？
3. 6.3 行程变化需要改期、提前离店或到店无房时，平台/酒店如何处理费用？
4. 6.4 找不到退改入口或退改被拒时，如何升级人工申诉，需要哪些凭证？

### 7. 支付担保押金发票

1. 7.1 酒店支持哪些支付方式（信用卡/外卡/企业卡/移动支付）？
2. 7.2 担保订单和到店付订单的留房时效、预授权规则是什么？
3. 7.3 押金收取与退回时效、发票开具方式分别如何？
4. 7.4 出现重复扣款、扣款成功但订单失败或发票信息错误时，怎么申诉？

### 8. 客房设施与商旅服务

1. 8.1 酒店 WiFi 是否全覆盖，视频会议稳定吗？
2. 8.2 房间空调、热水、门锁、电视投屏等设施配置如何？
3. 8.3 酒店是否有洗衣房、健身房、泳池、会议室等商旅设施？
4. 8.4 设施故障（空调/热水/WiFi/门锁）如何报修，多久上门，超时能否换房？
5. 8.5 房间隔音怎么样，能听到走廊说话声或楼上拖椅子声吗？
6. 8.6 泳池/温泉水质、开放时间和高峰人流情况怎么样？

### 9. 宠物与特殊需求

1. 9.1 酒店是否允许携带宠物，有无清洁费或限制？
2. 9.2 是否可安排无烟房、无障碍房、安静房等偏好？
3. 9.3 生日/蜜月布置、加被子枕头等特殊需求最晚什么时候提交？
4. 9.4 特殊需求到店未兑现时，能否换房、补偿或免费取消，找谁处理？

### 10. 位置周边与安全应急

1. 10.1 酒店到地铁/商圈/景点的交通便利度如何？
2. 10.2 周边便利店、药店、医院和夜间出行安全情况如何？
3. 10.3 周边施工噪音、装修异味、安静房型如何判断与规避？
4. 10.4 夜间噪音投诉、房卡丢失或突发身体不适时，酒店应急与投诉通道是什么？
5. 10.5 酒店周边吃饭和逛街方便吗，会不会太偏？
6. 10.6 酒店官方联系方式是什么（前台电话/公众号/官网）？
