# QA 问答库数据管理工具

这是一个基于 PySide6 的本地桌面工具，用来管理类似 `QA产品库.xlsx` 这种问答库 Excel 文件。

## 启动方式

双击：

```bat
run_qa_manager.bat
```

或在当前目录执行：

```powershell
.\.venv\Scripts\python.exe main.py
```

也可以直接带文件路径启动：

```powershell
.\.venv\Scripts\python.exe main.py "C:\path\to\QA产品库.xlsx"
```

## 主要功能

- 选择一个 Excel 问答库文件。
- 自动检测 A-D 列是否已经是完整明细结构。
- 如果仍有合并单元格或 A-D 空值，可以点击“标准化 A-D 列”，直接在当前 Excel 文件中取消合并并向下填充。
- 如果文件已经标准化，按钮会自动置灰。
- 按 `分类 -> 区域 -> POI -> 咨询意图` 浏览问答数据。
- 区域、POI、咨询意图支持模糊搜索。
- 点击咨询意图后显示全部问答对。
- 在已选 POI 的咨询意图下新增 POI 专属问答，专属问答不会进入标准问题模板。
- 选中某条具体问答后，可批量修改该分类下所有 POI 的同款问题回复。
- 选中问答后可以编辑回复内容，并点击“保存修改”写回 Excel。
- 支持新增、删除、重命名 POI。
- 支持按分类编辑问题模板，模板会同步到该分类下所有 POI。

## 写回规则

- 所有改动都只写入当前选择的 Excel 文件。
- 新增 POI 会套用该分类当前问题模板，回复内容为空。
- 删除 POI 会删除该 POI 下全部问答。
- 重命名 POI 会更新该 POI 下全部问答行。
- 模板新增的问题会同步到所有同分类 POI，回复内容为空。
- 模板删除的问题会从所有同分类 POI 中删除。
- 模板修改意图或问法时，会尽量保留原问法对应的回复内容。
- 数据管理页新增的 POI 专属问答只属于当前 POI，不参与模板同步，也不会套用到未来新增的 POI。
- 数据管理页的“批量修改回复”会按 `分类 + 咨询意图 + 具体问法` 精确匹配，把该分类下所有 POI 的同款问题回复统一覆盖为新回复。
- 程序会在 Excel 的隐藏 G 列写入 `__QA_MANAGER_TYPE` 元数据：标准模板行为 `template`，POI 专属问答行为 `custom`。因此后续更新并应用模板时，`custom` 问答会被保留，不会被模板同步抹除。
- GUI 中会隐藏咨询意图和具体问法开头的编号，例如 `1. 门票与优惠政策` 显示为 `门票与优惠政策`，Excel 底层原值仍按原文件保存。
